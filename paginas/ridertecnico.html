RIDER TÉCNICO

El proveedor de audio asegurará tener personal cualificado disponible para prestar la asistencia necesaria al STAFF, así como, un técnico de sonido encargado del equipo en cada una de las consolas, tanto en main como en monitores.


A continuación se enumera todo el material necesario para la realización del espectáculo:


  1. Mezcladora de 24 canales
  2. Mezcladora para monitores
  3. Cinco monitores
  4. Ingeniero y/o técnico de audio en consola
  5. Cinco micrófonos para batería
  6. Dos cajas directas (bajo y teclado)
  7. Dos micrófonos Sure SM57 para microfonear y dos amplificadores de guitarra (no línea)
  8. Un micrófono inálambrico Sure SM58
  9. Un micrófono Sure SM58 para coros guitarrista
